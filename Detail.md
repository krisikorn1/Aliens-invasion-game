# 👽 Alien Invasion Game

## 🎮 Overview
Alien Invasion is a Java-based endless runner game inspired by the Chrome Dinosaur game. Players control an alien character that runs continuously, jumps over obstacles, and collects coins to achieve the highest score possible.

## 🚀 Features
- Endless running gameplay  
- Jump mechanics to avoid obstacles  
- Randomized obstacle generation  
- Coin collection system  
- Increasing game speed over time  
- Distance tracking (e.g., 40km per second)  
- High score system  
- Game over screen with restart option  
- In-game shop with 10 purchasable items  

## 🛠️ Technologies Used
- Java  
- Object-Oriented Programming (OOP)  
- Java GUI (Swing/JavaFX depending on your implementation)

## 🎯 Gameplay
- Press to jump and avoid obstacles  
- Collect coins to increase your total  
- Survive as long as possible while speed increases  

## 📌 Purpose
This project was developed to practice Java programming, game logic, and object-oriented design while creating an interactive and engaging game experience.
